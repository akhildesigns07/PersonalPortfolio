# Akhil Spring Portfolio

Spring Boot portfolio website built with:
- Spring Boot 3
- Thymeleaf
- Java 21

## Run locally

```bash
./mvnw spring-boot:run
```

or if Maven is installed:

```bash
mvn spring-boot:run
```

Then open:

```text
http://localhost:8080
```

## Features
- Single-page portfolio website
- Resume download served from the application
- Thymeleaf-driven content from Java model/controller
- Easy to customize professional sections
